# Cataloging Reports

MetaDB queries for cataloging reports.

## InstancesNoHoldings.sql
- Generates a list of instance records with no holdings attached.

## LinkCountsbyPublicNote.sql
- Generates a count of instance records by 856 public note values.

## SearchMARCsubfield.sql
- Search a MARC subfield for specific content.

## Holdingsitemlocationmismatch.sql
- Generates a list of items in which the item permanent location does not match the holdings permanent location.
